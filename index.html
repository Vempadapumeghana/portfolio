Citywide OD Passenger Flow Prediction

Deep Learning Smart City

Objective

Accurately predict citywide Origin-Destination (OD) flow on sparse metro networks using an Adaptive Feature Fusion Network (AFFN).

Tech Stack

  • Backend: Python, Anaconda, Jupyter Notebook
  • Frontend: Flask, HTML5, CSS3, JavaScript, Bootstrap4
  • Database: Sqlite3
  • Architecture: EMGC-GRU, Multi-Task Learning

System Analysis

Feature Existing System Proposed AFFN System
Method Survey Campaigns / Manual Estimation Enhanced Multi-Graph Convolution (EMGC-GRU)
Efficiency Time-consuming and manual High-speed automated prediction
Accuracy Low due to static data High due to spatial-temporal fusion
